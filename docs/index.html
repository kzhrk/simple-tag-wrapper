wrap-with-tag

A wrap-with-tag provides a simple function to wrap string with html.

This HTML is before excuting the Function wrapWithTag.
<p>Company - FizzBuzz</p>
<p>!!company!!</p>
Execute the function after loading the DOM.
wrapWithTag({
  regexp: /(company)/gi,
  className: 'font',
  tagName: 'em'
});
This HTML is after excuting the Function wrapWithTag.

Strings that match the regular expression are wrapped with html <em class="font"></em>

<p><em class="font">Company</em> - FizzBuzz</p>
<p>!!<em class="font">company</em>!!</p>

If you want to define any property, you can use attr option.

Execute the function after loading the DOM.
wrapWithTag({
  regexp: /(company)/gi,
  className: 'font',
  tagName: 'em',
  attr: {
    tabindex: -1
  }
});
This HTML is after excuting the Function wrapWithTag.

Strings that match the regular expression are wrapped with html <em class="font"></em>

<p><em class="font" tabindex="-1">Company</em> - FizzBuzz</p>
<p>!!<em class="font" tabindex="-1">company</em>!!</p>
